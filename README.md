# 環境情報

| 機能 | バージョン |
| ---- | ---- |
| Linux/Ubuntu | 20.4.* |
| .NET | 6.0 |
| C# | .NET依存 |
| ASP.NET | 6.2.3 |


# 環境構築


## .NET環境のインストール

```bash
sudo apt-get update && sudo apt-get install -y dotnet6
```

プロジェクトの実行

```bash
dotnet run
```
